# doc README

`doc/` 目录已按“用户文档”和“开发文档”区分：

- `doc/` 根目录
  - 面向最终用户的协议配置实践文档
  - 当前包括 `TUIC`、`NaiveProxy` 两篇用户指南

- `guides/`
  - 面向开发者/维护者的部署、联调、操作指南
- `reference/`
  - 键值映射、字段速查等参考资料
- `design/`
  - 设计方案、重构设计、长期规划
- `analysis/`
  - 调研、补充分析、方案评估
- `implementation/`
  - MVP、实施规格、落地方案、维护说明
- `reports/`
  - 迭代总结、项目报告、最终总结
- `todo/`
  - 暂未实施的未来待办
